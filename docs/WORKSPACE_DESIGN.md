# 工作空间设计与零部件清单

面向低压配电柜柔性装配的三机械臂协同工作单元。

---

## 1. 总体布局

三机械臂 + 中央装配工位 + 四个功能区域 + 一个共享避碰区：

```
┌──────────────────────────────────────────────────┐
│              柔性装配工作空间                     │
│                                                  │
│  上料区 / 原料区           元件料仓区             │
│  ┌────────────┐         ┌────────────┐           │
│  │ 柜体底板   │         │ 电气元件   │           │
│  │ 安装板     │         │ 端子/模块  │           │
│  └────────────┘         └────────────┘           │
│       ↑                       ↑                  │
│     R1 机械臂               R2 机械臂             │
│   上料定位臂               元件装配臂              │
│                                                  │
│           ┌──────────────────────┐               │
│           │    中央装配工位       │              │
│           │   配电柜安装板        │              │
│           │   共享作业区域        │              │
│           └──────────────────────┘               │
│                    ↑                             │
│                 R3 机械臂                        │
│           锁螺丝 / 检测机械臂                     │
│                                                  │
│           检测区 / 成品下料区                     │
└──────────────────────────────────────────────────┘
```

## 2. 三台机械臂分工

| 机械臂 | 位置 | 职责 | 推荐末端 |
|--------|------|------|----------|
| R1 | 左侧 | 上料、搬运、安装板定位 | 夹爪 / 吸盘 |
| R2 | 右侧 | 元件抓取、元件装配 | 小型夹爪 |
| R3 | 中下方 | 螺丝锁付、视觉检测、返修 | 电批模型 + 相机模型 |

## 3. 功能区域划分

### 3.1 上料区

- 配电柜安装底板（A/B/C 型三种）
- 料盘、定位销、挡块
- 待装配工件托盘

### 3.2 元件料仓区

| 元件 | 表示方式 |
|------|----------|
| 断路器 | 长方体模块 |
| 接触器 | 较大长方体模块 |
| 继电器 | 小方块模块 |
| 端子排 | 长条形模块 |
| DIN 导轨 | 细长金属条 |
| 螺丝盒 | 盒子模型 |

注：前期用 3D 打印件、亚克力块或简单几何体代替真实电气元件。

### 3.3 中央装配工位（核心区域）

划分为四个子区：

| 子区 | 可进入机械臂 | 用途 |
|------|-------------|------|
| 装配区 A | R1 | 放置安装板 |
| 装配区 B | R2 | 放置元件 |
| 锁付区 C | R3 | 螺丝锁付 |
| 共享区 S | 多臂需申请区域锁 | 过渡和协同 |

### 3.4 锁付检测区

- 螺丝孔位标记点
- 电批末端模型
- 相机检测模型
- 合格/不合格状态标识

### 3.5 成品下料区

- 成品托盘
- 合格品 / 返修品分区

## 4. 产品类型定义（A/B/C 型柜）

| 型号 | 元件数量 | 螺丝数量 | 检测点数 | 工艺复杂度 |
|------|---------|----------|---------|-----------|
| A 型柜 | 3 | 6 | 2 | 简单 |
| B 型柜 | 5 | 10 | 3 | 中等 |
| C 型柜 | 6 | 12 | 4 | 较复杂 |

## 5. 实物搭建采购清单

### 5.1 基础平台

| 名称 | 建议 |

| 电批模型 | 3D 打印假模型 |

### 5.4 控制与传感器

| 名称 | 用途 |
|------|------|
| 工控机 / 笔记本 | 运行 ROS2、调度算法 |

| 限位开关 / 光电传感器 | 检测工件到位（可选） |

## 6. 最小可行版本（仿真优先）

| 必须有 | 可以简化 |
|--------|---------|
| 3 台机械臂（真机或仿真） | 螺丝用点位停留表示，不用真实拧紧 |
| 1 个中央装配工位 | 检测用状态判断代替真实视觉 |
| 上料区 + 料仓区 + 锁付检测区 | 线束用未来拓展表示 |
| A/B/C 三种产品任务 | 输送带用静态模型代替 |
| 3 类工艺：上料、装配、锁付/检测 | 不做复杂物理抓取仿真 |

## 7. 实施顺序

```
第 1 步：确定工作台尺寸 + 三台机械臂位置
第 2 步：建立中央装配工位模型
第 3 步：放置上料区、元件料仓区、检测区
第 4 步：定义每个区域的关键坐标点
第 5 步：建立 A/B/C 三类产品任务定义
第 6 步：实现单机械臂动作模板
第 7 步：实现三机械臂调度
第 8 步：加入共享区域锁避碰
第 9 步：加入急单插入和故障重分配
第 10 步：美化场景并录制视频
```

核心思路：先把工作空间抽象成「区域 + 点位 + 任务」，再逐步美化。
