# 五向测针标定误差补偿 EI 论文框架

版本：2026-06-15 初稿框架

## 拟定题目

中文题目：

基于标准球约束与分支感知高斯过程的机器人五向测针标定误差补偿方法

英文题目：

Branch-Aware Calibration Error Compensation for a Five-Direction Touch-Trigger Probe Using Sphere Constraints and Gaussian Process Residual Modeling

## 摘要框架

机器人搭载五向触发式测针可实现复杂工件的在线检测，但多分支红宝石球装配偏差、触发预行程、机器人绝对定位误差及横向分支误触会显著降低测量精度。针对五向测针分支间误差不可共用补偿的问题，本文提出一种基于标准球约束和分支感知高斯过程残差建模的标定误差补偿方法。首先，建立各分支红宝石球心相对机器人法兰的独立几何模型；其次，利用标准球触发数据构造方向约束和距离约束方程，并通过鲁棒最小二乘联合估计标准球球心与分支局部偏置；然后，依据残差、过行程和碰撞包络识别异常触发样本；最后，引入分支感知高斯过程回归对剩余空间残差进行小样本概率补偿，并输出预测不确定度以实现在线可信度门控。实验拟在 CR5 工业机器人、RMP60 触发式测头和五向测针平台上开展，对比名义几何、传统最小二乘、鲁棒标定和所提方法的补偿效果。预期结果表明，所提方法可降低五向测针跨分支测量误差，提高横向分支标定稳定性，并减少标准球补采次数。

关键词：

机器人在机测量；五向测针；触发式测头；标准球标定；高斯过程回归；误差补偿；主动标定

## 1 引言

### 1.1 研究背景

工业机器人在线测量正在替代部分专用量仪和离线检测流程。触发式测头具有结构简单、成本较低、可达性强等优点；五向测针进一步提高了侧壁、内腔和多方位特征的可测性。然而，五向测针不是一个单一 TCP，而是五个具有不同局部偏置和接触方向的红宝石球组合。若使用普通单测针标定方法，会导致跨分支误差传播。

### 1.2 现有研究不足

现有机器人标定与补偿研究可分为三类：

1. 参数化几何标定：通过 DH 参数、MDH 参数或工具偏置优化提高绝对定位精度，但对测针分支触发差异建模不足。
2. 非参数残差补偿：通过 RBF、SVR、GPR、Kriging 或神经网络建立空间误差映射，但通常忽略五向测针结构约束。
3. 主动标定和物理约束学习：通过不确定度或信息增益选择标定点，或者将机器人运动学嵌入网络模型，但现场小样本条件下仍需要明确的几何先验。

### 1.3 本文贡献

本文贡献拟写为：

1. 提出一种五向测针分支独立标定模型，将每个红宝石球心作为独立局部偏置求解，避免多分支共用 TCP。
2. 构建标准球方向约束和距离约束的两阶段标定流程，兼顾人工粗采和自动法向精采。
3. 设计基于残差统计、过行程和碰撞包络的异常触发识别方法，提升横向分支标定可靠性。
4. 提出分支感知 GPR 残差补偿方法，在小样本条件下输出误差补偿量和预测不确定度。
5. 给出主动补采策略，提高姿态覆盖效率，降低人工试错成本。

## 2 系统组成与问题定义

### 2.1 测量系统

实验平台：

1. 工业机器人：CR5 六轴机器人。
2. 测头：RMP60 触发式测头。
3. 测针：五向测针，包括 `z`、`x_pos`、`x_neg`、`y_pos`、`y_neg` 五个分支。
4. 标定件：20 mm 标准球，球半径 `R_s = 10 mm`。
5. 红宝石球：半径 `R_p = 1 mm`，依据 `config/cross_probe_geometry.yaml`。

### 2.2 数据定义

每条触发记录包括：

```text
branch, F_i, R_i, approach_i, trigger_state, stop_pose, overtravel, collision_margin
```

本项目已有相关数据：

1. `data/yneg_gui_normal_absolute_fit_20260610.json`：`y_neg` 单分支阶段性绝对标定结果，6 点 RMS 约 `0.058 mm`。
2. `data/five_branch_sphere_calibration_plan_20260612.csv`：五分支标准球采样计划。
3. `config/cross_probe_geometry.yaml`：五向测针名义几何模型。

### 2.3 目标

给定触发瞬间法兰位姿和触发分支，估计真实接触点：

```text
P_contact = f(F_i, R_i, branch_i, approach_i) - compensation(x_i)
```

并使补偿后的标准球重构误差、平面重构误差或工件特征误差最小。

## 3 标定与补偿方法

### 3.1 五向测针分支几何模型

第 `b` 个分支的局部红宝石球心偏置为：

```text
p_b = p_origin + d_b L_b + delta_b
```

其中：

1. `p_origin` 为分支公共名义起点。
2. `d_b` 为分支方向单位向量。
3. `L_b` 为名义分支长度。
4. `delta_b` 为该分支待估装配偏差。

触发瞬间球心：

```text
P_i,b = F_i + R_i p_b
```

### 3.2 标准球方向约束标定

在可靠接近方向已知时：

```text
r_i,b = C_s - F_i - R_i p_b - a_i (R_s + R_p)
```

优化目标：

```text
min sum rho(||r_i,b||_2) + lambda ||p_b - p_b0||^2
```

`rho(.)` 采用 Huber 或 Cauchy 损失，以降低异常点影响。

### 3.3 距离约束粗标定

人工粗采阶段，接近方向可能不稳定，采用距离约束：

```text
r_i,b^d = ||F_i + R_i p_b - C_s|| - (R_s + R_p)
```

该模型用于得到标准球球心和分支偏置初值，并驱动下一阶段自动法向触碰。

### 3.4 异常触发识别

定义异常分数：

```text
S_i = w1 ||r_i|| + w2 |overtravel_i| + w3 angle_error_i + w4 collision_penalty_i
```

样本满足以下条件之一时标记为异常：

1. 残差超过中位数绝对偏差阈值。
2. 接近方向与两球心连线夹角超过阈值。
3. 路径碰撞裕度低于安全阈值。
4. 停止过行程明显偏离同批样本统计范围。
5. 同姿态重复触发点离散度超过阈值。

该部分可解释本项目早期横向分支纯 `z-` 触碰产生毫米级异常残差的原因。

### 3.5 分支感知 GPR 残差补偿

基础标定后的残差：

```text
e_i,b = P_i,b^fit - P_i^reference
```

输入特征：

```text
x_i = [branch, F_i, R_i, a_i, overtravel_i]
```

对三维残差分别训练 GPR：

```text
e_k = GP(m_k(x), k_k(x, x')), k in {x, y, z}
```

核函数建议：

```text
k = k_SE(position) + k_Matern(direction) + k_rot(rotation) + k_branch(branch)
```

其中 `k_branch` 使同一分支样本强相关，不同分支样本弱相关或通过多任务核共享公共误差。

补偿：

```text
P_i^corr = P_i^raw - mu_GP(x_i)
```

可信度：

```text
U_i = trace(Sigma_GP(x_i))
```

若 `U_i` 超限，则提示补采或禁止自动出具高置信测量结果。

### 3.6 主动补采策略

候选采样点来自机器人可达姿态集合和分支安全接近方向集合。采样得分：

```text
Score(x) = alpha U_GP(x) + beta O(x) - gamma C(x)
```

其中：

1. `U_GP(x)` 为 GPR 预测不确定度。
2. `O(x)` 为标定雅可比矩阵可观测度增益。
3. `C(x)` 为碰撞风险和路径代价。

选择高不确定、可观测度高且安全的点进行补采。

## 4 实验设计

### 4.1 标定实验

实验步骤：

1. 对每个分支采集不少于 6 个标准球触发样本，建议 10 到 15 个。
2. 姿态覆盖不同 `RZ`、少量 `RX/RY`，横向分支避免非目标结构先接触。
3. 记录触发瞬间法兰位姿、接近方向、停止过行程和碰撞裕度。
4. 分别执行距离粗标定、方向精标定、鲁棒异常剔除和 GPR 残差建模。

### 4.2 验证实验

验证对象：

1. 标准球留出样本重构。
2. 平面或立方体侧面点阵测量。
3. 多分支测量同一几何特征的闭合一致性。

### 4.3 对比方法

1. Nominal：名义五向测针几何。
2. LS：普通最小二乘分支偏置标定。
3. Robust-LS：鲁棒分支偏置标定。
4. Robust-LS + GPR：本文主方法。
5. Robust-LS + RBF/SVR：传统机器学习对照。
6. Physics-informed NN：样本扩展后的物理约束神经网络对照。

### 4.4 评价指标

1. 标准球重构 RMS。
2. 最大残差和 P95 残差。
3. 分支内重复性。
4. 分支间一致性。
5. 异常识别准确率。
6. 主动补采后的误差下降率。
7. GPR 置信区间覆盖率。

## 5 结果与讨论框架

### 5.1 分支独立标定结果

表格建议：

```text
branch | samples | rank | condition | offset_x | offset_y | offset_z | RMS | max_error
```

重点讨论：

1. 每个分支偏置与名义几何的差异。
2. 姿态覆盖对条件数和 RMS 的影响。
3. 横向分支异常触碰对结果的影响。

### 5.2 补偿效果

表格建议：

```text
method | sphere_RMS | sphere_max | plane_RMS | branch_consistency | P95
```

预期现象：

1. 名义几何误差最大。
2. 普通 LS 可降低系统偏差，但受异常点影响明显。
3. 鲁棒 LS 稳定性更高。
4. GPR 对剩余空间残差进一步降低，且能输出不确定度。

### 5.3 消融实验

消融项：

1. 去掉分支独立建模。
2. 去掉鲁棒核。
3. 去掉碰撞裕度异常筛查。
4. 去掉方向核或姿态核。
5. 去掉主动补采。

### 5.4 工程适用性讨论

讨论内容：

1. 小样本条件下 GPR 相比深度网络更稳健。
2. 深度网络适合后续大规模数据积累后的跨工件、跨姿态泛化。
3. 预测不确定度可作为机器人在线检测质量追溯依据。
4. 标定数据需要覆盖实际测量姿态，否则应触发补采或降级为参数补偿。

## 6 结论

本文提出了一种面向机器人五向触发式测针的分支独立标定误差补偿方法。该方法通过标准球约束求解各分支局部红宝石球心偏置，利用鲁棒优化和异常触发识别提高横向分支标定可靠性，并采用分支感知 GPR 建立小样本残差补偿模型。该框架适用于机器人在机测量中五向测针多分支、多姿态和小样本标定场景。后续工作将扩展到热漂移、机器人柔顺误差和跨工件特征测量补偿。
