# 五向测针标定误差补偿方法专利框架

版本：2026-06-15 初稿框架

## 一、拟定名称

一种面向工业机器人五向触发式测针的分支独立标定与小样本残差补偿方法、系统及存储介质

备选名称：

1. 一种五向测针多分支球标定误差建模与在线补偿方法
2. 一种基于标准球约束和高斯过程残差场的五向测针测量误差补偿方法
3. 一种适用于机器人在机测量的五向测针分支自适应标定补偿系统

## 二、技术领域

本发明属于机器人在机测量、触发式测头标定、几何误差补偿和智能制造检测技术领域，具体涉及一种将标准球几何约束、五向测针分支独立参数辨识、鲁棒非线性优化、小样本高斯过程残差建模及在线不确定度门控相结合的测针标定误差补偿方法。

## 三、背景技术

工业机器人搭载触发式测头进行工件在线检测时，机器人具有较好的重复定位能力，但绝对定位精度受关节零偏、连杆参数误差、柔顺变形、末端工具装配误差、触发过行程和测针预行程变化等因素影响。对于五向测针，轴向测针与四个横向分支对应不同红宝石球，若将单一 TCP 或统一补偿量用于所有分支，会引入分支间系统误差。

现有常见方法包括：

1. 基于标准球的单测针 TCP 标定，仅求解一个工具中心点，难以覆盖五向测针多红宝石球结构。
2. 基于最小二乘的参数标定，可求得分支偏置，但对异常触碰、方向覆盖不足和机械臂姿态误差敏感。
3. 基于神经网络或深度学习的误差补偿，需要较大样本，难以直接适应现场小样本标定。
4. 基于机器人整体标定的误差补偿，通常未显式区分测针分支、接近方向和触发状态。

近年机器人标定领域出现了无迹卡尔曼滤波结合可变步长 LM、粒子滤波结合启发式优化、高斯过程主动学习、物理约束注意力残差网络等方法。这些方法表明，小样本、不确定度评估、物理约束和主动采样是高精度补偿的重要方向。本发明将上述思想限定到五向触发式测针实体结构和标准球标定流程中，形成可工程落地的补偿链路。

## 四、要解决的技术问题

本发明拟解决以下问题：

1. 五向测针每个分支的红宝石球中心相对法兰坐标系存在独立装配偏差，不能共用单一工具偏置。
2. 横向分支触碰标准球时容易发生测杆、中心结构或非目标分支先接触，导致异常样本混入。
3. 标准球标定数据往往为小样本，传统深度网络训练不稳定，常规最小二乘又缺乏局部残差泛化能力。
4. 测量补偿不仅需要给出补偿量，还需要判断当前姿态、接近方向和分支是否落在可信标定域内。
5. 在机器人在机测量中，需要将标定、路径规划、碰撞筛查、误差补偿和质量追溯整合为闭环流程。

## 五、总体技术方案

本发明提出一种“分支独立几何标定 + 鲁棒残差辨识 + 小样本概率补偿 + 在线门控”的方法。

系统由以下模块组成：

1. 五向测针几何先验模块：保存轴向及四个横向分支的名义方向、名义偏置、球半径、测杆半径和探头外形参数。
2. 标准球触发采集模块：记录触发瞬间法兰位置、姿态、分支编号、接近方向、触发状态、过行程估计和路径安全状态。
3. 分支独立绝对标定模块：以同一个标准球球心和各分支局部球心偏置为待估参数，进行鲁棒优化求解。
4. 异常触碰识别模块：根据球面约束残差、方向一致性、停止过行程、碰撞包络和残差统计量剔除异常样本。
5. 残差补偿模型模块：以分支、法兰姿态、接近方向、测量空间位置和过行程为输入，建立带物理约束的残差预测模型。
6. 主动补采模块：依据模型不确定度和参数可观测度，推荐下一组标定姿态和接近方向。
7. 在线补偿模块：对实际工件测量点进行分支选择、可信域判断、补偿量计算和不确定度输出。

## 六、核心数学模型

### 6.1 分支球心模型

对第 `b` 个分支、第 `i` 次触发，法兰位姿为：

```text
T_i = [R_i, F_i]
```

其中 `R_i` 为法兰姿态旋转矩阵，`F_i` 为法兰位置。第 `b` 个红宝石球心在法兰坐标系下的局部偏置为 `p_b`，则触发瞬间红宝石球心为：

```text
P_i,b = F_i + R_i p_b
```

若标准球球心为 `C_s`，标准球半径为 `R_s`，红宝石球半径为 `R_p`，接近方向单位向量为 `a_i`，理想触碰满足：

```text
C_s = F_i + R_i p_b + a_i (R_s + R_p)
```

或以距离形式表达：

```text
||F_i + R_i p_b - C_s|| = R_s + R_p
```

其中方向约束形式适合已知可靠接近方向的精标定，距离约束形式适合人工粗采阶段。

### 6.2 鲁棒联合估计

定义残差：

```text
r_i,b = C_s - F_i - R_i p_b - a_i (R_s + R_p)
```

对所有分支和样本构建目标函数：

```text
min {C_s, p_b} sum rho(||r_i,b||_2) + lambda sum ||p_b - p_b0||_W^2
```

其中 `rho(.)` 为 Huber、Cauchy 或 Tukey 鲁棒核，`p_b0` 为名义几何偏置，`lambda` 为先验约束权重。该项使小样本条件下的解不偏离五向测针物理结构。

### 6.3 残差补偿模型

在得到基础标定参数后，将每个样本的剩余误差表示为：

```text
e_i,b = P_measured_i,b - P_reference_i
```

输入特征定义为：

```text
x_i,b = [branch_onehot, F_i, q_i or R_i, a_i, overtravel_i, local_direction_b]
```

主补偿模型采用分支感知高斯过程回归：

```text
e_x, e_y, e_z ~ GP(m_b(x), k_b(x, x'))
```

核函数采用空间核、姿态核和方向核的乘积或加权和：

```text
k_b = k_space(F, F') + k_rot(R, R') + k_dir(a, a') + k_branch(b, b')
```

补偿后的测量点为：

```text
P_corrected = P_raw - E[e | x, D]
```

并输出预测方差：

```text
sigma_corrected^2 = sigma_probe^2 + sigma_robot^2 + sigma_GP^2
```

当 `sigma_GP` 超过阈值或输入点远离标定样本分布时，系统提示重新标定或主动补采。

### 6.4 主动补采准则

对候选姿态集合 `X_c`，计算参数可观测度和 GPR 不确定度：

```text
score(x) = alpha * tr(Sigma_GP(x)) + beta * cond(J)^-1 + gamma * clearance(x)
```

其中 `J` 为标定方程雅可比矩阵，`clearance(x)` 为碰撞安全项。选择得分最高且路径安全的候选点进行下一次标准球触碰。

## 七、算法流程

1. 建立五向测针名义模型，输入每个分支方向、名义偏置、球半径、测杆和探头外形。
2. 选择一个标准球，采集每个分支在多姿态、多接近方向下的触发数据。
3. 对人工粗采数据使用距离约束求解标准球球心和分支偏置初值。
4. 根据初值自动生成沿两球心连线的短行程法向触碰路径，并进行测杆和非目标分支碰撞筛查。
5. 对精采数据进行方向约束鲁棒最小二乘，得到每个分支的绝对局部红宝石球心偏置。
6. 计算残差并进行异常样本筛除，异常依据包括残差范数、停止过行程、方向夹角、碰撞裕度和同姿态重复性。
7. 基于有效样本训练分支感知 GPR 残差模型，得到残差均值和不确定度。
8. 对候选采样姿态进行不确定度评估，必要时主动补采。
9. 在工件测量时，根据触发分支和当前位姿计算原始接触点，再叠加残差补偿。
10. 输出补偿后的测量点、置信度、所用分支、补偿模型版本和标定数据版本，实现质量追溯。

## 八、与现有技术相比的有益效果

1. 将五向测针每个红宝石球视作独立分支，可避免多分支共用 TCP 造成的系统误差。
2. 通过标准球球心和分支偏置联合求解，可在不依赖高精度外部测量仪的情况下完成现场标定。
3. 鲁棒核函数和碰撞包络筛查能够识别横向分支非目标接触、纯 Z 向误触等异常情况。
4. GPR 适合小样本现场标定，能够同时给出补偿量和不确定度，比单纯最小二乘更适合在线可信度判断。
5. 主动补采使标定点覆盖高不确定区域，可减少人工试错和无效触碰次数。
6. 该方法将测针实体结构、机器人路径规划、标定求解和补偿模型耦合，具备工程实施性和专利可保护性。

## 九、实施例设计

### 实施例 1：单分支绝对标定

以 `y_neg` 横向分支为例，采集 6 到 15 个不同姿态下的标准球触发点。根据本项目当前数据，`data/yneg_gui_normal_absolute_fit_20260610.json` 已得到 6 点、满秩、RMS 约 `0.058 mm` 的阶段性结果，可作为实施例中的初始验证数据来源。

### 实施例 2：五分支联合标定

利用 `data/five_branch_sphere_calibration_plan_20260612.csv` 规划 `z`、`x_pos`、`x_neg`、`y_pos`、`y_neg` 五个分支的标准球触碰路径。每个分支至少采集 6 个方向覆盖充分的触发点，并分别求解 `p_b`。

### 实施例 3：残差补偿

用留一验证或分层交叉验证比较以下方法：

1. 无补偿：仅使用名义几何。
2. 参数补偿：仅使用分支独立最小二乘偏置。
3. 鲁棒参数补偿：使用鲁棒核和异常剔除。
4. GPR 残差补偿：在鲁棒参数补偿基础上叠加小样本概率残差。
5. 物理约束神经网络补偿：作为样本量扩展后的对照方法。

评价指标包括 RMS、最大残差、P95 残差、重复性、跨分支一致性、补偿不确定度覆盖率和主动补采效率。

## 十、权利要求书框架

### 权利要求 1：独立方法权利要求

一种面向工业机器人五向触发式测针的分支独立标定与误差补偿方法，其特征在于，包括：

1. 建立五向测针的分支几何先验模型；
2. 采集标准球触发数据，所述触发数据包括法兰位置、法兰姿态、触发分支、接近方向和触发状态；
3. 对每个分支分别构建红宝石球心模型；
4. 基于标准球几何约束联合求解标准球球心和各分支局部球心偏置；
5. 根据球面残差、路径碰撞裕度和过行程信息剔除异常触发样本；
6. 建立以分支、位置、姿态和接近方向为输入的残差补偿模型；
7. 对工件测量触发点进行分支匹配、残差预测和测量点坐标补偿；
8. 输出补偿后的测量点和补偿不确定度。

### 权利要求 2：分支独立模型

根据权利要求 1 所述的方法，其中第 `b` 个分支的红宝石球心由 `F_i + R_i p_b` 表示，`p_b` 为该分支独立局部偏置，不同分支不共用同一工具中心点。

### 权利要求 3：标准球约束

根据权利要求 1 所述的方法，其中标准球约束采用方向约束、距离约束或二者组合，方向约束为 `C_s = F_i + R_i p_b + a_i(R_s + R_p)`。

### 权利要求 4：鲁棒优化

根据权利要求 1 所述的方法，其中联合求解采用包含 Huber、Cauchy 或 Tukey 鲁棒核的非线性最小二乘，并加入分支名义几何先验约束。

### 权利要求 5：异常识别

根据权利要求 1 所述的方法，其中异常触发样本依据球面残差、接近方向夹角、停止过行程、同姿态重复性、非目标分支碰撞包络中的至少两种指标进行识别。

### 权利要求 6：GPR 残差补偿

根据权利要求 1 所述的方法，其中残差补偿模型为分支感知高斯过程回归模型，输出三维残差均值和预测方差。

### 权利要求 7：主动补采

根据权利要求 1 所述的方法，还包括根据残差模型不确定度、标定雅可比矩阵条件数和路径安全裕度生成下一标准球触碰姿态。

### 权利要求 8：在线可信度门控

根据权利要求 1 所述的方法，其中当待测点输入特征超出标定样本覆盖范围或预测方差超过阈值时，系统禁止自动补偿或提示重新标定。

### 权利要求 9：系统权利要求

一种五向测针标定误差补偿系统，包括：数据采集模块、分支几何建模模块、标准球标定模块、异常识别模块、残差补偿模块、主动补采模块和在线测量补偿模块，所述模块用于执行权利要求 1 至 8 任一项所述方法。

### 权利要求 10：存储介质权利要求

一种计算机可读存储介质，其上存储有程序，所述程序被处理器执行时实现权利要求 1 至 8 任一项所述方法。
