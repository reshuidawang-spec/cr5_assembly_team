# 专利交底书初稿：五向测针标定误差补偿方法

版本：2026-06-15

## 1 发明名称

一种面向工业机器人五向触发式测针的分支独立标定与误差补偿方法、系统及存储介质

## 2 技术领域

本发明涉及工业机器人在机测量、触发式测头标定、五向测针几何误差建模、机器人末端工具坐标补偿和智能制造质量检测技术领域，尤其涉及一种基于标准球几何约束、分支独立参数辨识、鲁棒异常触发剔除、高斯过程残差补偿和在线不确定度门控的五向测针标定误差补偿方法。

## 3 背景技术

工业机器人搭载触发式测头可用于工件在线检测、加工后尺寸复核、夹具定位校验和复杂曲面采样。与传统三坐标测量机相比，机器人在机测量具有部署灵活、可达空间大、可与生产线集成等优点。然而，工业机器人通常重复定位精度较好而绝对定位精度有限，测量结果受到关节零偏、连杆参数误差、末端法兰装配误差、工具中心点误差、结构柔顺变形、触发式测头预行程变化和控制器停止过行程等因素影响。

五向测针由轴向测针和四个横向分支组成，五个红宝石球分别对应不同接触方向。与单向测针不同，五向测针不能简单视为一个固定工具中心点。每个红宝石球受制造、装配和连接结构影响，其球心相对法兰坐标系的局部偏置均不相同。若将某一分支标定得到的工具中心点直接用于其他分支，会导致跨分支系统误差。

现有技术主要存在以下不足：

1. 单测针 TCP 标定方法通常只求解一个工具中心点，不能准确描述五向测针各分支的独立红宝石球心。
2. 基于普通最小二乘的标准球标定方法对异常样本敏感，横向分支触碰时容易发生测杆、中心结构或非目标分支先接触，从而产生毫米级误差。
3. 纯参数标定只能修正固定几何偏置，对机器人姿态相关误差、接近方向相关触发误差和停止过行程残差建模不足。
4. 神经网络等深度学习补偿方法通常需要大量样本，而现场标准球标定往往是小样本场景。
5. 现有补偿方法较少输出补偿可信度，难以判断某一测量姿态是否位于标定样本覆盖范围内。

近年来，工业机器人误差补偿研究出现了可变步长 Levenberg-Marquardt、无迹卡尔曼滤波、粒子滤波、高斯过程回归、主动学习和物理约束神经网络等方向。这些方法分别提高了参数寻优抗噪能力、小样本残差建模能力和采样效率。但是，如何将上述思想与五向触发式测针的多分支实体结构、标准球触碰几何约束和在线测量流程结合，仍缺少可直接工程实施的方案。

## 4 发明目的

本发明的目的在于提供一种面向工业机器人五向触发式测针的标定误差补偿方法，用于解决五向测针多分支共用 TCP、横向分支误触、现场小样本补偿和在线补偿可信度不足的问题。

本发明希望达到以下效果：

1. 对五向测针五个红宝石球分别建立独立局部偏置模型。
2. 利用标准球触碰数据联合求解标准球球心和各分支球心偏置。
3. 对异常触碰样本进行识别和剔除，提高横向分支标定稳定性。
4. 在分支独立几何标定基础上建立小样本残差补偿模型。
5. 对每次补偿输出预测不确定度，支持在线测量可信度判断。
6. 根据不确定度和可观测性自动推荐补采姿态，减少人工试错。

本发明的核心发明点可概括为：将五向测针的五个红宝石球从“一个工具中心点”改为“五个分支独立球心”，并将标准球几何约束、异常触发识别、小样本概率残差补偿和主动补采组合为一个闭环系统。该组合使补偿结果既具有几何可解释性，又能覆盖机器人姿态和触发方向引入的非线性残差。

### 5.1 系统组成

本发明的五向测针标定误差补偿系统包括：

1. 几何先验模块，用于保存五向测针结构参数，包括轴向分支、四个横向分支、红宝石球半径、测杆半径、分支名义方向和分支名义长度。
2. 触发采集模块，用于采集标准球触发瞬间的机器人法兰位置、法兰姿态、接近方向、触发分支、触发时间、停止位姿和过行程信息。
3. 路径安全模块，用于根据测头外形、测杆包络和标准球位置判断目标分支触碰路径是否存在非目标结构碰撞风险。
4. 分支标定模块，用于基于标准球几何约束求解各分支局部红宝石球心偏置。
5. 异常识别模块，用于依据球面残差、接近方向一致性、过行程、重复性和碰撞裕度剔除异常触发样本。
6. 残差补偿模块，用于建立分支感知的残差预测模型，输出三维补偿量和预测不确定度。
7. 主动补采模块，用于根据残差模型不确定度、标定雅可比矩阵条件数和路径安全裕度生成下一组标准球触碰姿态。
8. 在线补偿模块，用于在实际工件测量时读取触发分支和法兰位姿，计算补偿后的测量点坐标并输出置信信息。

上述模块既可部署在机器人上位机，也可部署在与机器人控制器通信的测量工作站。触发采集模块应保证触发瞬间法兰位姿与 DI 触发信号时间一致，在线补偿模块应保存补偿模型版本、标定数据版本、触发分支和预测不确定度，以满足测量结果追溯。

### 5.2 分支独立几何模型

设工业机器人第 `i` 次触发时法兰坐标系相对于机器人基坐标系的位姿为：

```text
T_i = [R_i, F_i]
```

其中 `R_i` 为三维旋转矩阵，`F_i` 为三维位置向量。

五向测针包含分支集合：

```text
B = {z, x_pos, x_neg, y_pos, y_neg}
```

对任一分支 `b`，其红宝石球心在法兰坐标系下的局部偏置为：

```text
p_b = p_b0 + delta_b
```

其中 `p_b0` 为名义几何偏置，`delta_b` 为待估装配误差。触发瞬间该分支红宝石球心在机器人基坐标系下为：

```text
P_i,b = F_i + R_i p_b
```

本发明明确限定不同分支对应不同 `p_b`，不允许五个分支共用同一个工具中心点。

五向测针分支独立几何模型如图 2 所示。每个分支的局部偏置 `p_b` 可以由名义偏置 `p_b0` 与待估装配误差 `delta_b` 表示，轴向分支与横向分支均参与统一的数据格式和补偿流程，但参数互不混用。

### 5.3 标准球方向约束

设标准球球心为 `C_s`，标准球半径为 `R_s`，测针红宝石球半径为 `R_p`。当第 `i` 次触发的接近方向单位向量为 `a_i`，且目标分支红宝石球先接触标准球时，理想几何关系为：

```text
C_s = F_i + R_i p_b + a_i (R_s + R_p)
```

由此定义三维残差：

```text
r_i,b = C_s - F_i - R_i p_b - a_i (R_s + R_p)
```

对多个分支和多次触发样本，构建鲁棒优化目标：

```text
min sum rho(||r_i,b||_2) + lambda sum ||p_b - p_b0||_W^2
```

其中 `rho(.)` 为鲁棒核函数，优选 Huber 核、Cauchy 核或 Tukey 核；`lambda` 为名义几何先验权重；`W` 为各方向权重矩阵。

标准球方向约束和距离约束如图 3 所示。方向约束用于自动法向精采阶段，距离约束用于人工粗采阶段。二者可以在同一标定流程中级联使用，也可根据现场采集质量单独使用。

![图3 标准球方向约束与距离约束](assets/fig3_sphere_constraint.svg)

### 5.4 标准球距离约束粗标定

对于人工粗采阶段，接近方向可能不完全可靠。此时采用距离约束建立初值：

```text
||F_i + R_i p_b - C_s|| = R_s + R_p
```

残差定义为：

```text
r_i,b^d = ||F_i + R_i p_b - C_s|| - (R_s + R_p)
```

距离约束结果作为标准球球心和分支偏置初值，不直接作为最终补偿参数。得到初值后，系统根据两球心连线生成短行程法向触碰路径，进入精标定阶段。

### 5.5 异常触发识别

由于五向测针横向分支容易发生非目标接触，本发明设计异常触发识别步骤。对第 `i` 个样本计算异常评分：

```text
S_i = w1 ||r_i,b|| + w2 |o_i| + w3 theta_i + w4 c_i + w5 q_i
```

其中：

1. `||r_i,b||` 为球面几何残差。
2. `o_i` 为停止过行程。
3. `theta_i` 为接近方向与两球心连线方向夹角。
4. `c_i` 为碰撞风险项，由非目标分支、测杆和探头外壳到标准球的最小裕度计算。
5. `q_i` 为同姿态重复性离散度。

当 `S_i` 超过阈值，或单项指标超过安全阈值时，将样本标记为异常，不参与最终分支参数求解和残差补偿训练。

### 5.6 小样本残差补偿

基础几何标定后，仍可能存在由机器人姿态、触发预行程、过行程和局部空间非线性误差造成的剩余误差。本发明采用分支感知高斯过程回归进行小样本残差补偿。

输入特征为：

```text
x_i = [branch_i, F_i, R_i, a_i, o_i]
```

输出残差为：

```text
e_i = [e_x, e_y, e_z]
```

对三个坐标方向分别建立高斯过程模型：

```text
e_k ~ GP(m_k(x), k_k(x, x')), k in {x, y, z}
```

核函数由位置核、姿态核、方向核和分支核组成：

```text
k = k_position + k_rotation + k_direction + k_branch
```

补偿后的测量点为：

```text
P_corrected = P_raw - mu_GP(x)
```

其中 `mu_GP(x)` 为高斯过程残差均值。预测不确定度为：

```text
U(x) = trace(Sigma_GP(x))
```

当 `U(x)` 超过阈值，系统提示当前测量点超出标定可信域，需重新标定、补采或降低测量结果置信等级。

### 5.7 主动补采

系统在候选姿态集合中计算采样得分：

```text
Score(x) = alpha U_GP(x) + beta G(J_x) - gamma Risk(x)
```

其中：

1. `U_GP(x)` 为残差模型预测不确定度。
2. `G(J_x)` 为候选点对标定雅可比矩阵条件数或信息矩阵行列式的改善量。
3. `Risk(x)` 为路径碰撞风险、可达性代价和非目标接触风险。

系统优先选择不确定度高、可观测性增益大且路径安全的姿态进行下一次标准球触碰。

## 6 有益效果

与现有技术相比，本发明具有以下有益效果：

1. 分支独立：每个红宝石球具有独立局部偏置，避免五向测针共用 TCP 造成的系统误差。
2. 工程可实施：仅依赖标准球和机器人触发数据，不强制要求激光跟踪仪等昂贵外部测量设备。
3. 抗异常：通过残差、过行程、方向一致性和碰撞包络识别非目标触碰。
4. 小样本适应：采用高斯过程残差建模，适合现场少量标准球标定样本。
5. 可解释：几何标定参数具有明确物理意义，残差补偿和不确定度可追溯。
6. 可扩展：可进一步引入机器人柔顺误差、热漂移、关节角特征和物理约束神经网络。
7. 可闭环：主动补采模块能根据不确定度自动推荐下一采样姿态，减少人工试错。

## 8 具体实施方式

以下结合 CR5 工业机器人、RMP60 触发式测头、五向测针和 20 mm 标准球说明本发明的具体实施方式。该实施例仅用于说明本发明，并不限制本发明保护范围。

### 8.1 建立名义几何模型

系统读取五向测针几何配置。以本项目配置为例：

1. 标准球半径 `R_s = 10 mm`。
2. 红宝石球半径 `R_p = 1 mm`。
3. 分支集合为 `z`、`x_pos`、`x_neg`、`y_pos`、`y_neg`。
4. 四个横向分支名义方向分别为正负 X、正负 Y，轴向分支为正 Z。
5. 分支名义长度为 75 mm。

每个分支的名义局部偏置从五向测针结构文件读取，并作为后续鲁棒优化先验。

表 1 给出一个实施例中的关键几何参数。

| 参数 | 符号 | 示例值 | 说明 |
| --- | --- | ---: | --- |
| 标准球半径 | `R_s` | `10 mm` | 20 mm 标准球 |
| 红宝石球半径 | `R_p` | `1 mm` | 五向测针球半径 |
| 分支名义长度 | `L_b` | `75 mm` | 分支起点到球心距离 |
| 分支数量 | `|B|` | `5` | z、x_pos、x_neg、y_pos、y_neg |
| 触发位姿 | `F_i, R_i` | 实测 | 由机器人反馈记录 |
| 接近方向 | `a_i` | 单位向量 | 由路径规划或两球心连线确定 |

### 8.2 单分支粗标定

操作者选择一个目标分支，例如 `y_neg`。机器人在多个姿态下使该分支红宝石球触碰标准球，采集触发瞬间法兰位姿。由于人工粗采阶段接近方向不一定可靠，系统先使用距离约束求解标准球球心和 `y_neg` 分支偏置初值。

### 8.3 自动法向精采

系统根据粗标定得到的标准球球心 `C_s` 和分支偏置 `p_b`，实时计算当前红宝石球心：

```text
P_i,b = F_i + R_i p_b
```

然后计算两球心连线方向：

```text
a_i = normalize(C_s - P_i,b)
```

系统沿该方向规划短行程触碰路径，并在执行前检查：

1. 目标红宝石球是否为最近接触对象。
2. 测杆、探头主体和非目标分支是否存在碰撞风险。
3. 路径起点、接触点和过行程目标点是否均有安全裕度。

确认安全后执行低速触碰，记录触发瞬间数据。

横向分支安全筛查如图 5 所示。对于 `x_pos`、`x_neg`、`y_pos` 和 `y_neg` 等横向分支，系统优先规划沿目标红宝石球与标准球球心连线的短行程触碰路径，并避免纯竖直方向下压造成的非目标分支或测杆接触。

![图5 横向分支异常触发与路径安全筛查](assets/fig5_collision_screening.svg)

### 8.4 分支绝对标定

系统对有效样本使用方向约束鲁棒最小二乘求解 `C_s` 和 `p_b`。本项目已有 `y_neg` 阶段性数据，6 个样本满秩，RMS 残差约 `0.058 mm`，可作为实施例的预实验结果。该数据说明在姿态覆盖和触碰方向较合理时，分支独立标准球标定能够得到稳定参数。

### 8.5 五分支标定

在单分支流程验证后，系统按照五分支标准球采样计划依次采集：

```text
z, x_pos, x_neg, y_pos, y_neg
```

每个分支建议采集不少于 6 个有效样本，优选 10 至 15 个样本。对每个分支分别求解局部偏置 `p_b`，并统一保存标准球球心、分支参数、残差、样本状态和模型版本。

### 8.6 残差补偿训练

对于已完成鲁棒标定的样本，系统计算基础几何标定后的剩余残差。以分支编号、法兰位置、法兰姿态、接近方向和停止过行程作为输入，训练分支感知 GPR 模型。若样本数量不足，则仅启用参数补偿；当样本数量达到阈值后，启用 GPR 残差补偿。

分支感知高斯过程残差补偿模型如图 6 所示。模型输入不仅包含空间位置，还包含触发分支、姿态、接近方向和过行程，从而表达分支相关误差、方向相关触发误差和机器人姿态相关残差。

![图6 分支感知高斯过程残差补偿模型](assets/fig6_gpr_compensation.svg)

### 8.7 工件在线测量

测量工件时，系统在触发瞬间读取法兰位姿和触发分支，计算原始红宝石球心和接触点，再调用残差补偿模型：

```text
P_corrected = P_raw - mu_GP(x)
```

若预测不确定度小于阈值，则输出补偿后的测量点；若不确定度超过阈值，则输出低置信状态，并提示补采或重新标定。

在线补偿与不确定度门控如图 7 所示。该过程能够防止系统在标定样本未覆盖的姿态或空间区域内直接输出高置信测量结果。

### 8.9 算法伪代码

```text
输入：五向测针名义几何 G，标准球半径 R_s，红宝石球半径 R_p
输出：各分支偏置 p_b，残差补偿模型 M，在线补偿点 P_corrected

1. 读取五向测针分支集合 B 与名义偏置 p_b0
2. 对每个分支 b in B:
   2.1 人工粗采标准球触发样本 D_b0
   2.2 用距离约束求解 C_s 与 p_b 初值
   2.3 根据 C_s 与 p_b 生成法向触碰路径
   2.4 执行路径安全筛查，剔除高风险候选姿态
   2.5 自动精采标准球触发样本 D_b
   2.6 用方向约束和鲁棒核求解 p_b
   2.7 根据残差、过行程和碰撞裕度剔除异常样本
3. 汇总有效样本，计算基础标定后的剩余残差 e_i
4. 若样本数量满足阈值，训练分支感知 GPR 模型 M
5. 在线测量时，根据 branch_i, F_i, R_i 计算 P_raw
6. 由 M 输出 μ_GP(x_i) 与 Σ_GP(x_i)
7. 若 trace(Σ_GP) 小于阈值，输出 P_corrected = P_raw - μ_GP
8. 否则提示主动补采或降低测量置信等级
```

## 9 权利要求书草案

### 权利要求 1

一种面向工业机器人五向触发式测针的分支独立标定与误差补偿方法，其特征在于，包括以下步骤：

1. 建立五向触发式测针的分支几何先验模型，所述分支几何先验模型包括轴向分支和四个横向分支的名义方向、名义局部偏置和红宝石球半径；
2. 采集标准球触发数据，所述标准球触发数据包括触发瞬间机器人法兰位置、法兰姿态、触发分支、接近方向和触发状态；
3. 对每个触发分支分别建立红宝石球心模型，所述红宝石球心模型由机器人法兰位置、法兰姿态和该分支局部球心偏置确定；
4. 基于标准球几何约束联合求解标准球球心和各触发分支局部球心偏置；
5. 根据球面残差、接近方向一致性、停止过行程和路径碰撞裕度识别异常触发样本；
6. 基于有效触发样本建立分支感知残差补偿模型；
7. 在工件在线测量时，根据当前触发分支和机器人法兰位姿计算原始测量点，并利用所述分支感知残差补偿模型对原始测量点进行补偿；
8. 输出补偿后的测量点坐标及其预测不确定度。

### 权利要求 2

根据权利要求 1 所述的方法，其特征在于，第 `b` 个分支红宝石球心在机器人基坐标系下的位置表示为：

```text
P_i,b = F_i + R_i p_b
```

其中 `F_i` 为第 `i` 次触发瞬间机器人法兰位置，`R_i` 为机器人法兰姿态旋转矩阵，`p_b` 为第 `b` 个分支在法兰坐标系下的局部球心偏置。

### 权利要求 3

根据权利要求 1 所述的方法，其特征在于，所述标准球几何约束包括方向约束，所述方向约束为：

```text
C_s = F_i + R_i p_b + a_i (R_s + R_p)
```

其中 `C_s` 为标准球球心，`a_i` 为接近方向单位向量，`R_s` 为标准球半径，`R_p` 为红宝石球半径。

### 权利要求 4

根据权利要求 1 所述的方法，其特征在于，所述标准球几何约束还包括距离约束，所述距离约束为：

```text
||F_i + R_i p_b - C_s|| = R_s + R_p
```

所述距离约束用于人工粗采阶段求解标准球球心和分支局部球心偏置初值。

### 权利要求 5

根据权利要求 1 所述的方法，其特征在于，联合求解标准球球心和分支局部球心偏置时采用包含鲁棒核函数和分支名义几何先验项的非线性最小二乘目标函数。

### 权利要求 6

根据权利要求 1 所述的方法，其特征在于，异常触发样本根据球面残差、接近方向与两球心连线夹角、停止过行程、同姿态重复性和非目标结构碰撞裕度中的至少两种指标进行识别。

### 权利要求 7

根据权利要求 1 所述的方法，其特征在于，所述分支感知残差补偿模型为高斯过程回归模型，其输入包括触发分支、机器人法兰位置、机器人法兰姿态、接近方向和停止过行程，其输出包括三维残差均值和预测方差。

### 权利要求 8

根据权利要求 1 所述的方法，其特征在于，当所述预测不确定度超过预设阈值时，系统提示当前测量点超出标定可信域，并执行重新标定、主动补采或降低测量置信等级中的至少一种操作。

### 权利要求 9

根据权利要求 1 所述的方法，其特征在于，还包括主动补采步骤，所述主动补采步骤根据残差补偿模型预测不确定度、标定雅可比矩阵可观测性和路径碰撞风险选择下一标准球触碰姿态。

### 权利要求 10

一种面向工业机器人五向触发式测针的分支独立标定与误差补偿系统，其特征在于，包括几何先验模块、触发采集模块、路径安全模块、分支标定模块、异常识别模块、残差补偿模块、主动补采模块和在线补偿模块，所述系统用于执行权利要求 1 至 9 任一项所述方法。

### 权利要求 11

一种计算机可读存储介质，其上存储有计算机程序，所述计算机程序被处理器执行时实现权利要求 1 至 9 任一项所述方法。

## 10 摘要草案

本发明公开了一种面向工业机器人五向触发式测针的分支独立标定与误差补偿方法、系统及存储介质。该方法首先建立五向测针轴向分支和四个横向分支的独立红宝石球心模型，采集标准球触发瞬间机器人法兰位置、姿态、接近方向和触发分支信息；然后基于标准球方向约束和距离约束联合求解标准球球心及各分支局部球心偏置；再根据球面残差、接近方向一致性、停止过行程和路径碰撞裕度识别异常触发样本；最后利用分支感知高斯过程回归建立小样本残差补偿模型，对工件在线测量点进行补偿并输出预测不确定度。本发明能够避免五向测针多分支共用工具中心点造成的系统误差，提高横向分支标定稳定性，并为机器人在机测量提供可追溯的补偿可信度。
